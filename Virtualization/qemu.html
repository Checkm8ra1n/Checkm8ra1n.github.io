QEMU - The cross platform command line emulator

QEMU

QEMU is a free and open source command line emulator for make VMs

Price: free

Avaible Platforms:

How to install on Windows

How to install on macOS

You have to install Macports or Homebrew

Command for Macports

sudo port install qemu

Command for Homebrew

brew install qemu

How to install on Linux

Command for Debian/Ubuntu

sudo apt install qemu-system

Command for Arch

pacman -S qemu

Command for Fedora

dnf install @virtualization

Command for suse

zypper install qemu

How to make VM

Since that QEMU is a command line tool, you have to make the virtual HDA and power the VM with the Terminal or CMD
