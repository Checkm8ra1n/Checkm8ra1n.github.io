SideStore


SideStore is a fork (modified version) of AltStore that for sideload and refresh apps, needs only a wifi connection, so without USB

Now on your iDevice go to the AppStore and download WireGuard; direct link -> https://apps.apple.com/us/app/wireguard/id1441195209?ls=1 and this file: WireGuard config file

Go to the SideStore official website and download the .iPA of SideStore on your PC: SideStore Site or download it directly from there -> SideStore iPA

From this github page, download jitterbug for your PC OS: https://github.com/osy/Jitterbug/releases/

Now you have to install AltServer on your PC (for sideload the Sidestore iPA), so download AltServer if you don't have it (direct link -> https://altstore.io/#Downloads)

When you will open AltStore, make sure to hold the Alt or Option ‚å• key on your keyboard, you should see the option "sideload .ipa" instead of "Install AltStore"

Now when the Finder or Explorer window will appaer, select the SideStore iPA that you donwloaded before

Before opening SideStore, open the Terminal/CMD where you have donwloaded the jitterbug executable, run it and it will generate the mobiledevicepairing file. Once generated, you have to copy that file in your iDevice, so AirDrop it or send it via USB sharing or with LocalSend

Now open the WireGuard config file, this will automatically configure the SideStore VPN

Once you open SideStore, will ask you the mobiledevicepairing file, select it

(if the file doesn't work, try to change the file extension from .mobiledevicepairing to .plist)

go to My Apps section, make sure that you are conncted to a Wi-Fi Network and you will are able to install iPAs without PC!

Note: SideStore will only work with Wi-Fi, not with 5G! Beacuse anisette data can be trasmetted with Wi-Fi only
