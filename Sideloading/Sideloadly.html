Sideloadly

Sideloadly is the best software for sideload Apps on iDevices using a computer (macOS and/or Windows), so here I will show you how to use this powerful tool


First Download Sideloadly from the official website: https://sideloadly.io/

Note for Windows: you have to Download iTunes and iCloud

(not from Microsoft Store, if you have them, uninstall them)

  • On Windows simply execute the Installer that you downloaded

  • On macOS mount the dmg and drag and drop the Sideloadly app on Applications folder (or whatever you want)

  1. Now you have to connect your iDevice to your computer

    2. Then you have to drag and drop the IPA file that you want to install on your iDevice or click the IPA icon on Sideloadly and select the IPA from explorer/finder

  2. Enter the ID Apple on the box as the image above

  3. Click the Start button

Now Sideloadly will ask you your ID Apple password and (if you have it) the 2 factor authentication code

After that, the app will start the installation process

When the installation is done, you will see the app on your iDevice

Now once the App is installed, go to settings, Privacy & Security, scroll on the bottom of the page and click on developer mode tab and enable the toggle of Developer Mode

One you enabled the Developer Mode, go to settings again, General, VPN & Device Management, click on you apple ID and click trust

Enjoy you app :)
