Preparing Modded Nintendo Switch SD Card

1- Requirements

  • MicroSD card (at least 64GB recommended)
  • SD card reader (usually included with the MicroSD card)
  • A Windows, macOS, or Linux computer
  • Hakate (custom firmware bootloader)
    • Download hekate
  • Atmosphère (custom firmware)
    • Download Atmosphère

2- Installing the Bootloader

To install the custom firmware, once you downloaded Hakate and Atmosphere, extract the contents of both ZIP files to the root of your SD card.

3- Installing the Signature Patches

For make the games work, also if you payed them, you need to install the signature patches. These patches allow you to run homebrew applications and games that are not signed by Nintendo.

  • Download the signature patches
    • from here (wait the page loads completly)
  • Extract the contents of the ZIP file to the root of your SD card, merging the folders when prompted.
  • Download the hekate_ipl.ini file and replace with the original one on your SD card on the bootloader folder
    • Download here
  • Download also the syspatches, and copy all contents to the root of the SD card
    • Download here

4- Blocking Nintendo's Anti-Piracy Measures

For don't make Nintendo detect your custom firmware, you need to block their anti-piracy measures. This can be done by using the sysmodules and patches you downloaded earlier.
