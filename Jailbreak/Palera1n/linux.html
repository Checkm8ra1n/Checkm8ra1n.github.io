Linux is a great alternative if you don't have a mac, first open Terminal and copy the below command:

sudo /bin/sh -c "$(curl -fsSL https://static.palera.in/scripts/install.sh)"

Now simply plug-in your iDevice

Note: USB-C cables can create issues when injecting palera1n payloads. Use USB-A cable.

Make sure that you have trusted your mac

If your computer doesn't detect your iDevice in normal mode, install usbmuxd with apt or your package manager and run in another terminal window sudo systemctl stop usbmuxd and sudo usbmuxd -p -f just don't close that windows once you have finished to jailbroke you iDevice

Now you have to run palera1n and arguments, see here the list of commands and arguments to use or use my Palera1n GUI
